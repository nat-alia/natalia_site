Astro

Typeface Design & Website

2019

Capstone Research: Form and Interaction

A web-based interactive typeface built with HTML, CSS, and Javascript. I played with constantly changing states and experimented with parallax movement according to user mouse position. Through hovering the user can change the size of the modules, creating a sense of dimension within each letter. The website is currently a work in progress, but you can see a peak of it below.

Astro has been since adapted to a static typeface, where I focused on experimenting with the repetitive texture of the outlined modules.
